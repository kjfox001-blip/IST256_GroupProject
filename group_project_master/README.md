# Example Website

This example website uses the React framework on a Vite template to display an understanding about how to integrate JavaScript functionality, HTML, and Bootstrap elements into a relatively nice looking document. We chose historical arms and Armour due to the shear variety and the ease in which to get pictures that follow certain aesthetics.
